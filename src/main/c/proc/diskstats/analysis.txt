/proc/diskstats 파일 분석

1. 구성
각 행은 한 디스크나 파티션에 대한 정보
각 행은 여러 필드로 구성

2. 주요 항목
- 03번째 필드: 디스크 장치 이름

- 04번째 필드: 완료된 읽기 요청의 수
- 06번째 필드: 읽은 섹터의 수
- 07번째 필드: 읽기 소요 시간(ms)

- 08번째 필드: 완료된 쓰기 요청의 수
- 10번째 필드: 쓴 섹터의 수
- 11번째 필드: 쓰기 소요 시간(ms)

- 12번째 필드: 진행중인 I/O 작업의 수
- 13번째 필드: 진행 중인 I/O 작업 시간(ms)
- 14번째 필드: I/O 작업의 가중 시간(ms)

* 전체 시스템 I/O vs 실제 물리 디스크 I/O 성능
전체 시스템 I/O 성능 - 가상 디스크와 실제 디스크 장치 모두 측정
실제 물리 디스크 I/O 성능 - 실제 디스크 장치만 측정

이름이 loop로 시작하는 장치는 가상 디스크 장치
이름이 nvme로 시작하는 장치는 실제 디스트 장치
